MyMediaPlayer

A stable, multi-threaded universal media player.

Documentation

This application is a lightweight media player built using Python, Tkinter, and the FFmpeg engine (ffpyplayer). It is designed to handle high-resolution video and audio without interface hanging.

Key Controls

  • Open: Browse your system for any supported media file.
  • Pause/Play: Toggle playback. Note: The UI remains responsive even during heavy decoding.
  • Seek (-10s / +10s): Quickly jump through media using relative seeking.

Supported Formats

Video: MP4, MKV, AVI, MOV, WEBM Audio: MP3, WAV, FLAC, OGG, M4A Images: JPG, PNG, BMP, TIFF
