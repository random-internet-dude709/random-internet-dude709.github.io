Registry Editor

Version 1.0 • 2025

A modern, dark-themed Windows registry viewer and editor

Overview

This application is a lightweight, standalone alternative to regedit.exe with a modern user interface.

Main features

  • Dark theme (Windows 11 style)
  • Frameless window with custom title bar
  • Tree-like navigation with folder icons
  • Always-visible (Default) value for every key
  • Create / modify / delete values (most common types)
  • Create subkeys (not directly under hive roots)
  • Delete subkeys (with confirmation)
  • Permission elevation prompt when needed
  • Custom icon support

Installation & Usage

Requirements

  • Windows 10 or Windows 11 (64-bit)
  • No Python installation required (standalone .exe)

How to run

  1. Double-click regedit.exe
  2. Important: For editing HKEY_LOCAL_MACHINE, HKEY_CLASSES_ROOT and similar protected areas, right-click the executable → Run as administrator
  3. Use the path bar or double-click keys to navigate
  4. Right-click in the list to create/modify/delete items
Warning: Incorrect changes in the registry can make your system unstable or unbootable. Always back up important keys before modifying them.
