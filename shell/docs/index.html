Project Documentation

Modern Shell is a cross-platform terminal emulator built with Python and PySide6. It provides a sleek, dark-themed GUI wrapper around system shell commands, utilizing the qtmodern library for a native, sophisticated look.

Key Features

  • Asynchronous Execution: Uses QThreadPool to prevent the GUI from freezing during long-running tasks.
  • Modern UI: Frameless window support with dark mode styles.
  • Smart Command Handling: Detects interactive commands (cmd, powershell) and launches them in dedicated native windows.
  • Directory Tracking: Persistent Current Working Directory (CWD) management with a visual status bar.

Technical Architecture

The application follows a Worker-Signal pattern:

  1. Main Thread: Handles the PySide6 event loop and UI rendering.
  2. Worker Thread (QRunnable): Executes subprocess.run() or subprocess.Popen() to interact with the OS.
  3. Signals: Custom WorkerSignals bridge the gap between threads to update the console safely.
