ZipEncryptor Documentation

This utility is a high-security CLI tool designed to archive folders into ZIP files using AES-256 bit encryption.

How to Use

Run the executable from your terminal (Command Prompt or PowerShell) using the following syntax:

zipencrypt <input_folder> <output_zip_file> <password>

Example:

script.exe "C:\Documents\Private" "C:\Backups\Secret.zip" "MySafePassword123"
Compatibility Note: While this tool creates standard ZIP files, the built-in Windows "Compressed Folders" extractor in older versions of Windows may not support AES-256. For best results, use 7-Zip or WinRAR to decrypt files.
