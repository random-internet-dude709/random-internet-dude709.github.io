User Guide

Welcome to World Browser. This application is designed to be a lightweight, privacy-focused portal to the web.

  • Navigation: Use the top address bar to enter URLs. Press Enter to go.
  • Interaction: Right-click on pages to access standard context menus (Copy, Paste, Inspect).
  • Status: The bar at the bottom indicates page loading progress.

Technical Documentation

This browser is compiled using Nuitka and utilizes the following stack:

  • Core: Python 3.12
  • UI Framework: PyQt6
  • Rendering Engine: QtWebEngine (based on Chromium)
  • Architecture: Standalone x64 Binary
